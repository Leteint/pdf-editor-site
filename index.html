Vous pouvez désactiver la licence depuis le menu Aide → Licence, puis la réactiver sur votre nouveau poste. Le nombre de postes simultanés dépend de votre formule.
Les mises à jour futures sont-elles incluses ? +
Oui. Toutes les mises à jour sont incluses dans votre achat. Pas de frais supplémentaires.
L'application fonctionne-t-elle hors connexion ? +
Oui. Une vérification de licence est effectuée au démarrage, mais en cas d'absence de connexion, une période de grâce de 7 jours permet d'utiliser l'application normalement.
Sur quels systèmes fonctionne PDF Editor ? +
PDF Editor fonctionne sur Windows 10 et Windows 11 (64-bit). Une version Mac/Linux n'est pas prévue pour l'instant.
Comment fonctionne l'intégration N8N / Make ? +
Lorsque vous créez un formulaire PDF avec PDF Editor, les données saisies sont embarquées dans le fichier au format JSON. N8N peut lire ce JSON directement via un trigger "Watch Files" ou un nœud HTTP. Aucun plugin supplémentaire n'est nécessaire.

Oui. Lemon Squeezy génère automatiquement une facture conforme à la réglementation européenne après chaque achat. Elle est disponible dans votre email de confirmation.
