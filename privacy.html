Politique de Confidentialité

Dernière mise à jour : 6 avril 2026 · Conforme RGPD

🛡️ En résumé : PDF Editor traite vos fichiers PDF entièrement en local sur votre ordinateur. Nous ne voyons pas vos documents. Les seules données transmises à l'extérieur sont votre clé de licence et un identifiant technique de machine lors de la validation — rien d'autre.

1. Responsable du traitement

Jean-Baptiste Leteinturier, développeur indépendant, France.
Contact : GitHub Issues

2. Données collectées par le logiciel

L'application PDF Editor elle-même collecte un minimum absolu de données :

Donnée Finalité Stockage Transmise ?
Clé de licence Validation de l'achat Local (fichier chiffré sur votre PC) Oui — vers API Lemon Squeezy uniquement
Identifiant machine Lier la licence à un appareil Local uniquement Oui — vers API Lemon Squeezy uniquement (hash anonymisé)
Date de dernière validation Permettre l'utilisation hors ligne (7 jours) Local uniquement Non
Vos fichiers PDF Traitement par le logiciel Local uniquement Jamais

L'identifiant machine est un hash SHA-256 calculé à partir d'informations techniques de votre ordinateur (nom de machine, architecture). Il ne permet pas d'identifier une personne physique.

3. Données collectées par le site web

Le site web leteint.github.io/pdf-editor-site est hébergé par GitHub Pages. GitHub peut collecter des données de navigation standards (adresse IP, navigateur, pages visitées) conformément à sa propre politique de confidentialité.

Nous n'utilisons aucun cookie de tracking, aucun pixel publicitaire, aucun outil d'analytics sur ce site. Le seul stockage local utilisé est localStorage pour mémoriser votre préférence de langue — cette information ne quitte jamais votre navigateur.

4. Données collectées lors de l'achat

Les achats sont traités par Lemon Squeezy (Merchant of Record). Lors d'un achat, Lemon Squeezy collecte :

  • Votre adresse email
  • Vos informations de paiement (traitées directement par Stripe — nous n'y avons jamais accès)
  • Votre pays (pour le calcul de TVA)

Ces données sont soumises à la politique de confidentialité de Lemon Squeezy. Nous recevons uniquement votre adresse email pour la gestion de votre licence.

5. Base légale du traitement (RGPD)

  • Exécution du contrat — validation de licence, gestion des remboursements
  • Intérêt légitime — prévention de la fraude (vérification des activations)

6. Durée de conservation

  • Données de licence : conservées tant que la licence est active
  • Email lié à l'achat : conservé par Lemon Squeezy selon leurs règles (minimum 5 ans pour obligations comptables)
  • En cas de remboursement et révocation de licence : les données locales sur votre PC ne sont pas supprimées automatiquement — vous pouvez les supprimer manuellement dans le dossier %USERPROFILE%\.pdf_editor\

7. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès — connaître les données que nous détenons sur vous
  • Droit de rectification — corriger des données inexactes
  • Droit à l'effacement — demander la suppression de vos données
  • Droit à la portabilité — recevoir vos données dans un format structuré
  • Droit d'opposition — vous opposer à un traitement

Pour exercer ces droits : contactez-nous via GitHub Issues. Nous répondrons dans un délai de 30 jours.

Vous pouvez également introduire une réclamation auprès de la CNIL (Commission Nationale de l'Informatique et des Libertés) : cnil.fr

8. Sécurité

Les données de licence stockées localement sont chiffrées. Les communications avec l'API Lemon Squeezy se font exclusivement via HTTPS. Nous ne stockons aucun mot de passe ni donnée bancaire.

9. Modifications

Cette politique peut être mise à jour. La date de dernière modification est indiquée en haut de page. Les modifications substantielles seront communiquées via le dépôt GitHub.
